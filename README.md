# Описание
Это тестировочный репозиторий консольного "Морского боя". Сам код находится по следующему пути: 
Код отвечающий за интерактивное размещение кораблей игроком: Pozition/Progtam.cs
Код отвечающий за рандомное размещение кораблей противника: Тестовый Проект/Program.cs
Код постоянно меняется, поэтому меняется и функционал.
## Дополнительные эллементы
Программа использует текстовые файлы с изображениями карт, или просто длинным описанием.
Местоположения существующих на данный момент файлов: 
Карта(map.txt) = Тестовый Проект/bin/Debug/map.txt
Описание кораблей(ships.txt) = Тестовый проект/bit/Debug/map.txt
